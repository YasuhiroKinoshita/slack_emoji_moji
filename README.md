Slack Emoji moji
----------------

# Slack（じゃなくていいけど）のEmojiで字を書く

# usage

```
git clone https://github.com/YasuhiroKinoshita/slack_emoji_moji
cd slack_emoji_moji
bundle install --path vendor/local
bundle exec ruby 524.rb
```

# query

```
http://your-host-name/絵文字にしたい文字列（ただしa-zA-Z0-9の範囲）
```
